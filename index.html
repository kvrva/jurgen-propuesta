Procesamiento de Archivos Pesados

Servidor Optimizado para Carga Pesada

El portal estará integrado con un script PHP optimizado (`upload.php`) que maneja cargas por partes (chunks), evitando caídas de tiempo de espera (`timeout`) al subir archivos CAD grandes.

Resguardo Automático en Google Drive

Las cotizaciones y sus archivos planos DWG adjuntos se respaldan de manera inmediata en una estructura de carpetas privada de Google Drive mediante la API de Google, protegiendo la confidencialidad.

PLAN DE IMPLEMENTACIÓN

Hoja de Ruta hacia Producción

Cronograma detallado y entregables por fase para transformar esta propuesta interactiva en un portal web 100% operativo.

01
Semana 1

Fase 1: Infraestructura & Base CMS

Configuración del entorno definitivo de hosting en servidor virtual optimizado (VPS) y levantamiento del sistema base.

Tareas Principales:
  • Provisión de Servidor VPS (Vultr / DigitalOcean)
  • Instalación de Nginx/LiteSpeed y MariaDB
  • Levantamiento de WordPress Core
  • Maquetación inicial de Estilos y Responsive
02
Semana 2

Fase 2: Catálogo & Pasarela de Pagos (E-Commerce)

Implementación de WooCommerce, carga y clasificación del inventario de repuestos y vinculación de pagos reales.

Tareas Principales:
  • Configuración e-commerce y reglas de IVA (19%/Exento)
  • Integración de Pasarela Transbank Webpay Plus o Flow
  • Importación de repuestos y control de stock inicial
  • Automatización de correos de confirmación de compra
03
Semanas 3-4

Fase 3: Portal de Cotizaciones & Integración Drive API

Desarrollo del backend para procesamiento de archivos pesados y sincronización automática de planos en la nube corporativa.

Tareas Principales:
  • Subida segmentada de archivos CAD/PDF en PHP (Chunks)
  • Configuración de Credenciales de API de Google Drive
  • Script de subida automática Drive y orden de carpetas
  • Alertas por correo SMTP al equipo de Jürgen Von Arnim
04
Semana 5

Fase 4: Roles, Seguridad Activa & Lanzamiento

Implementación de permisos diferenciados de personal, fortificación del servidor, SEO básico y entrega final.

Tareas Principales:
  • Restricción de accesos por roles en panel de administración
  • Configuración DNS Cloudflare (HTTPS, WAF y Caché)
  • Instalación de Wordfence y respaldos automatizados
  • Entrega de credenciales, documentación y capacitación

Diseñado para el Crecimiento (+10.000 Clientes)

El sistema base de WordPress y base de datos MySQL/MariaDB cuenta con indexación optimizada en columnas de búsqueda rápidas (RUT, Email y Código de Cotización) asegurando tiempos de respuesta menores a 5ms. El panel administrativo incluye paginación dinámica del lado del servidor para evitar lentitud en el navegador, y permite la integración bidireccional mediante API REST con el ERP corporativo de Jürgen SpA (SAP, Defontana, etc.) para centralizar la facturación y el inventario.
